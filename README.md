Chúng tôi sẽ giới thiệu về app Ghi Chú Siêu Tốc beta 1.1
1.  Mở Đầu (Flow Khởi Động)
Khi bật app: Sẽ hỏi bạn muốn mở file đã có (Yes) hay tạo dữ liệu mới (No).
Yes: Chọn file JSSON hiện có → truy cập dữ liệu ngay.
No: Chọn thư mục lưu → app tạo file JSSON mới tại vị trí đã chọn.

2.  Quản Lý Ghi Chú (Core Features)
Tạo, Lưu / Xóa Ghi Chú.
Tải lại dữ liệu (refresh) khi cần.
Có Chế độ đọc để bro xem mà không sợ chỉnh sửa nhầm.
Lưu tất cả thông tin vào file JSSON. (Cái này là mấu chốt bảo mật đó bro).

3. Metadata & Tracking (Dấu Vết)
Ghi lại Ngày tạo ghi chú.
Ghi lại Thời gian chỉnh sửa cuối cùng (Hiển thị ngay trên tiêu đề luôn).

4. Tìm Kiếm & Sắp Xếp (Intelligence)
Tìm kiếm cực mạnh bằng #tag hoặc Đề.
Có thể ghim ghi chú quan trọng lên đầu.
Cho phép dán link trực tiếp vào ghi chú.
Đề Mục và Tags phân tách bằng dấu phẩy để quản lý rõ ràng.

5.Cá Nhân Hóa (Customization)
Chọn font hiển thị cho dễ đọc.
Không giới hạn dòng trong mục chính.

6.  Offline & Free (Cam Kết)
Hoàn toàn offline, không cần mạng.
Không quảng cáo.
Miễn phí 100%.

7. Chính Sách & Bảo Mật (Security First)
Tất cả ghi chú, metadata, tags, đề mục được lưu toàn bộ trên file JSSON → An toàn, riêng tư, người dùng toàn quyền kiểm soát.
